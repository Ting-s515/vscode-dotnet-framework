背景

  1. IDE 有區分紫色的 Visual Studio 和藍色的 Visual Studio Code，兩者都是 Microsoft 的產品，但它們是不同的開發工具。
  2. 紫色的 Visual Studio 是微軟專門開發 .net framework 的 IDE
  3. 但紫色的 Visual Studio 沒支援跨平台，加上 IDE 的體積龐大，我超級討厭
  4. 平常我只使用藍色的 Visual Studio Code，輕量、跨平台，可透過安裝套件來支援不同的程式語言和框架
  5. 主要是用於公司專案windows server 的 sharepoint 桌面開發，能夠支援部署、打包等功能
  6. 不需要支援WinForms / WPF / razor等

問題

  1. 目前市場套件是否有可讓 Visual Studio Code 支援 .net framework 的套件？
  2. 如果有，是否能提供一些推薦的套件名稱和使用方法？
  3. 如果沒有，是否有其他替代方案可以讓我在 Visual Studio Code 中開發 .net framework 的應用程式？或是自己開發一個套件來支援 .net framework？
